# Backend — Laravel scaffold (placeholder)

This folder contains a minimal scaffold for a Laravel-based backend that will provide Pterodactyl API compatibility.

To create a full Laravel app here, run inside this folder (on a Linux/macOS dev machine or inside the `panel` container):

```bash
composer create-project --prefer-dist laravel/laravel .
cp ../.env.example .env
php artisan key:generate
composer require guzzlehttp/guzzle
```
